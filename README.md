# Dentory 🦷

실제 진료 경험을 바탕으로 한 솔직한 리뷰를 통해,
믿을 수 있는 치과 정보를 제공하는 서비스입니다.

---

## 프로젝트 소개

Dentory는 광고성 정보보다 실제 이용자의 경험을 중요하게 생각합니다.

사용자는 주변 치과를 찾고,
후기를 확인하며,
치과 관련 이야기를 자유롭게 나눌 수 있습니다.

신뢰할 수 있는 치과 정보를 더 쉽게 찾을 수 있는 서비스를 목표로 개발하고 있습니다.

현재는 치과 정보 조회, 후기 조회, 커뮤니티 기능을 중심으로 1차 버전을 개발하고 있으며, 추후 실제 진료 경험을 기반으로 한 리뷰 시스템으로 확장하는 것을 목표로 하고 있습니다.

---

## 서비스 소개

### 치과 정보

위치, 진료 정보, 후기를 기반으로
주변 치과를 탐색할 수 있습니다.

### 후기 기반 정보

다른 사용자들의 경험을 참고하여
치과 선택에 도움을 받을 수 있습니다.

### 커뮤니티

치과 치료 경험과 궁금한 점을
다른 사용자들과 자유롭게 공유할 수 있습니다.

---

## 주요 기능

### 홈

* 슬라이드 배너
* 치과 찾기

### 치과

* 치과 리스트 조회
* 치과 검색
* 치과 상세 정보 조회
* 리뷰 조회

### 커뮤니티

* 게시글 목록 조회
* 게시글 상세 조회
* 게시글 작성, 수정, 삭제
* 댓글 등록 및 삭제
